# QUOV - Librería de Segunda Mano

## 📚 Sobre QUOV
QUOV es una librería de libros de segunda mano verificados en Ciudad de México.

## 🚀 Tecnologías
- Frontend: HTML, CSS, JavaScript, Tailwind CSS
- Backend: Node.js, Express
- IA: Anthropic Claude API

## 🛠️ Instalación

### Frontend
```bash
npm install
npm run build
